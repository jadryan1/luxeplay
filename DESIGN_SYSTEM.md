# Luxe Play NY - Design System

## Typography

### **Raleway** *(Google Fonts — the only font family used)*

| Role | Weight | Style |
|---|---|---|
| Display / Hero callouts | ExtraLight (200) | Large size, uppercase, wide `letter-spacing` |
| H1 — Large Headings | SemiBold (600) | Uppercase, wide `letter-spacing` |
| H2 — Secondary Headings | Regular (400) | Uppercase, tracked `letter-spacing` |
| H3 — Subheadings | Regular (400) | Smaller size, uppercase, wide `letter-spacing` |
| Body Copy | Regular (400) | Normal `letter-spacing`, mixed case |

- All headings are uppercase with generous letter-spacing
- Body copy is mixed case for readability
- Overall feel: minimal, soft, and elevated

## Color Palette

### **Primary Colors**
- **Pink (Backgrounds)**: `#F8DADA`
  - *Used in*: Hero Section Background, Header Navigation Background
- **Charcoal (Primary Text)**: `#2C2C2C`
  - *Used in*: Main Headings, Body Text, Logo "Luxe Play" text, Navigation Links

### **Accent Colors**
- **Gold**: `#D4AF37`
  - *Used in*: Logo "NY" text, Navigation Hover Underline, Eyebrow Text
- **Secondary Pink**: `#F4C2C2`
  - *Used in*: Dropdown Menu Hover
- **Rose/Light Pink**: `#FFB5D5`
  - *Used in*: Social Media Icons, Button Gradients
- **Transparent Pink**: `rgba(248, 218, 218, 0.95)`
  - *Used in*: Secondary Navigation Bar Background

### **Neutrals**
- **White**: `#FFFFFF`
  - *Used in*: Header Top Bar Background, Card Backgrounds
- **Medium Gray (Brownish)**: `#927474`
  - *Used in*: Subheadlines, Secondary Text
- **Black**: `#000000`
  - *Used in*: Image Placeholders

### **Gradients**
- **Pastel Rainbow**: `linear-gradient(90deg, #FFB5D5, #FFE5B4, #C8E6C9, #B3D9FF)`
  - *Used in*: "BOOK NOW" Button, "I'M READY!" Hero Button
